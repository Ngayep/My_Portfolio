About This Project

This project was inspired by the need for better tools to visualize public health data. By using this dashboard, researchers and public health officials can quickly analyze the spread of diseases like Malaria or AIDS in real-time, helping them make informed decisions.

This is a Portfolio Project developed for ALX Africa School. You can view the source code on GitHub and connect with me on social media.
